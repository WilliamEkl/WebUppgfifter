# CV-uppgift
Detta är ett CV som Jag har skapat med HTML, CSS och JavaScript. Syftet med detta projekt är att visa upp min kompetens och erfarenhet på ett visuellt tilltalande sätt.         

# Funktioner
CV:t innehåller:
- En ren och enkel design som alla kan förstå
- En responsiv layout
- Interaktiva element som låter dig ändra delar av designen

# Teknik som används
Projectet är skapat med hälp av följande tekniska hjälpmedel:
- HTML
- CSS
- JavaScript

# Hur man använder CV:t
För att använda detta CV, måse du helt enkelt öppna index.html filen i din webbläsare. Utöver det så finns det inte så mycket mer. På webbsidan kommer du se mitt cv med information och knappar längst ner på pappret som ger dig som användare möjlight att ändra utsendet på CV:t

# kundanpassning
Det finns två sätt att anpasa CV:t som du vill. Den första och enklaste är såklart att använda de knapper som har skapats på CV:t. Den andra är att gå in på koden och ändra utsendet till hur du själv vill att CV:t ska se ut.
